# Men Matar L Matar Mobile

Expo app for iOS and Android.

## Run locally

```bash
npm.cmd install
npm.cmd run start
```

Set `EXPO_PUBLIC_API_BASE_URL` to point to another backend when needed:

```bash
$env:EXPO_PUBLIC_API_BASE_URL="http://localhost:3000"; npm.cmd run start
```

The default API base URL is `https://menmatarlmatar.ma`.
